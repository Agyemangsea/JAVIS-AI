# 🤖 Javis AI - Natural Language Processing Assistant

A conversational AI that greets you throughout the day on your tablet using natural language processing and voice recognition.

## ✨ Features

- **Morning Greetings** - Personalized greeting when you wake up
- **Noon Check-ins** - Reminders and greetings during midday
- **Evening Chat** - Friendly evening interactions
- **Voice Recognition** - Listens to your spoken commands
- **Text-to-Speech** - Responds with natural voice output
- **Time & Date** - Tells you current time and date
- **Command Processing** - Understands natural language requests
